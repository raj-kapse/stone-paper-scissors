# Stone Paper Scissors

A simple terminal game written in Python. The player chooses stone, paper, or scissors and plays against the computer.

## How to Run

Make sure Python is installed, then run:

```bash
python stone_paper_scissors.py
```

## Example

```text
Choose one: stone, paper, or scissors
Enter your choice: stone
Your choice: stone
Computer choice: scissors
You win!
```
